This page covers how to use the [Querit](https://querit.com/) search API within LangChain. Querit is a reliable web search API providing comprehensive search functionality.

## Benefits for LangChain Users

The langchain-querit plugin bridges the gap between LLMs and real-time web data:

- **Solves LLM Knowledge Gaps**: Access to around 300 billion web pages in real-time addresses LLM knowledge cutoff limitations and ensures responses contain current, accurate information
- **Sub-second Latency**: Ultra-low response times enable agents to retrieve and process information instantly without impacting user experience
- **Enterprise-Grade Reliability**: High concurrency support ensures consistent performance under production workloads with reliable availability
- **High-Quality Sources**: Industry-leading algorithms and LLM understanding deliver precise, authoritative results from trusted global sources
- **Tailored Retrieval**: Advanced filtering by country and language allows customization for specific business scenarios and regional requirements
- **Drop-in Integration**: Built on LangChain's BaseTool pattern for seamless integration with existing agents, chains, and workflows
- **Structured Output**: Consistently formatted SearchResult objects (title, link, snippet, display_link, position) for easy parsing and downstream use
- **Simple Setup**: Minimal configuration required - just install the package and set QUERIT_API_KEY environment variable
- **Developer-Friendly Pricing**: 1,000 free queries per month with immediate access upon registration and zero-barrier integration into your AI projects
- **International Coverage**: Ideal choice for non-English developers with native support for 10 languages including Spanish, Portuguese, and more

## Quick Start

### Installation

```bash
pip install langchain-querit
```

### Basic Usage

```python
import os

# Set your Querit API key
os.environ["QUERIT_API_KEY"] = "your-querit-api-key"

# Initialize the WebSearchTool
from langchain_websearch import WebSearchTool
search_tool = WebSearchTool()

# Perform a search
results = search_tool._run("What is the capital city of France?")
print(results)
```

## Configuration

### Environment Variables

- `QUERIT_API_KEY`: Your Querit Search API key (required)

### WebSearchTool Parameters

- `num_results`: Number of results to return (default: 10, range: 1-50)
- `region`: Search region/language (default: "en-US")
- `safe_search`: Enable safe search filtering (default: True)

### Advanced Configuration

```python
from langchain_websearch import WebSearchTool

# Configure with specific parameters
search_tool = WebSearchTool(
    num_results=5,      # Number of results to return
    region="en-US",     # Search region
    safe_search=True    # Enable safe search
)

# Use with custom query
results = search_tool._run("machine learning tutorials")
print(results)
```
